# AWSについて

AWSとは、「Amazon Web Services」の通称であり、Amazonが提供するクラウドサービスである。

AWSのように、インターネットを経由してサービスを提供するシステムは「クラウドコンピューティング」と呼ばれる。

クラウドコンピューティングでは、「サーバー機器を購入する」「管理する」「スペースを確保する」「納品を待つ」といった必要がなく、インターネットからクラウドコンピューティングサービスに接続するだけで、必要なサーバーなどのリソースをすぐに利用できる。

AWSには多くのサービスが存在するが、代表的な4つのサービスとして以下のものがある。

## 代表的なAWSサービス

- **サーバー環境構築**  
  柔軟に設定変更できる仮想サーバーを作成・利用できるサービス（Amazon EC2）

- **データ保存・コンテンツ配信**  
  オンラインストレージとして利用でき、静的なコンテンツを配信できるサービス（Amazon S3）

- **データベース利用**  
  フルマネージドで管理負担を抑えられるデータベースサービス（Amazon RDS）

- **プログラムの実行環境**  
  サーバーの運用負担なしで、任意のプログラムを実行できる環境を提供するサービス（AWS Lambda）

## AWS認定クラウドプラクティショナーについて

今回受験するAWS認定クラウドプラクティショナーとは、AWS認定制度の1つであり、AWSやクラウドに関する基礎レベルの知識を認定する資格である。

AWS認定資格には3年間の有効期限が設けられており、日々進化するITおよびクラウド技術に対応するため、継続的に知識をアップデートする必要がある。
